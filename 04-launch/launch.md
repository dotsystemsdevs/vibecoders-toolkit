# 04 Launch

Going from closed testing to production.

---

## Before launch

- [ ] 100+ testers in closed testing
- [ ] Major bugs fixed
- [ ] Store listing ready
- [ ] Screenshots polished
- [ ] Privacy policy URL

---

## Store listing

### App name
- Keep it short
- Include keyword
- No jargon

### Description
```
Line 1: What it does (hook)
Line 2-3: Key features
Line 4: CTA
```

### Screenshots
- Show the app in use
- Add text overlays
- First screenshot = most important

---

## ASO (App Store Optimization)

| Element | Impact |
|---------|--------|
| App name | High |
| Screenshots | High |
| Description | Medium |
| Reviews | High |

---

## Launch day

- [ ] Submit for review
- [ ] Prepare social posts
- [ ] Tell your testers
- [ ] Post in niche communities
- [ ] Monitor crash reports

---

## After launch

- [ ] Respond to reviews
- [ ] Fix critical bugs fast
- [ ] Plan next update
- [ ] Track retention

---

*Launch is the beginning, not the end.*
