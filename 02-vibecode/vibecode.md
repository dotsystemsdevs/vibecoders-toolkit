# Vibecode

AI-generated code. Fast for MVPs.

```
Describe → Generate → Review → Test → Repeat
```

---

## Tools

| Tool | What | Use it? |
|------|------|---------|
| **Cursor** | AI VS Code | ✅ Best |
| **Claude Code** | Terminal AI | ✅ Complex tasks |
| **Windsurf** | AI editor | ✅ Alternative |
| **Bolt.new** | Full app gen | ❌ Skip |

---

## Models

| Model | Best for |

## Rules

- [ ] Plan first
- [ ] Be specific
- [ ] Review everything
- [ ] Test constantly
- [ ] Know when to stop

---

## Works vs Doesn't

| Works | Doesn't |
|-------|---------|
| Demos, MVPs | Production with users |
| Landing pages | Complex backends |
| Internal tools | Payment systems |

---

## Evidence

Shipped 3 apps in 1 week each using Claude + Cursor. Zero backend, all offline. That's the sweet spot.

Add backend/auth/payments → slow down and review everything.
